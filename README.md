# 🎮 Tiki Topple – Ancient Totem Duel

An interactive 2-player strategy game built using **HTML, CSS, and JavaScript**, where players compete to control a mystical totem pole and achieve their secret objectives.

---

## 🚀 Features

* 🎮 Strategic 2-player gameplay
* 🧾 Player name-based gameplay system *(can be added via input page)*
* 🧩 Unique mechanics like:

  * Topple (send tiki to bottom)
  * Toast (eliminate tiki)
  * Move up/down actions
* 🎨 Modern animated UI with glowing effects
* 🔊 Background music with toggle control
* 📱 Responsive design

---

## 🧑‍🤝‍🧑 Player Setup

Before starting the game, players can enter their names on a **dedicated name input page**.

These names are used for:

* Turn display
* Score tracking
* Winner announcement

*(Currently default names: Arkon & Zephyra — customizable)*

## 🎯 Gameplay

* Each player gets a **secret objective**
* Use action cards to manipulate the totem
* Score points based on top 3 tiki positions
* First player to reach **21 points wins** 🏆
